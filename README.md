# Todo CLI
A basic command Line Interface (CLI) To-do list application in Python, which allows displaying, adding and removing tasks by simple commands.

## How To Run?
1. Clone the repository.

2. Navigate to the project folder: cd To-do-List

3. Run commands:
   - python todo.py add "Task"
   - python todo.py remove "Task"
   - python todo.py display
  
# Note: 
tasks.txt is ignored by Git and can be created automatically while adding the first task.
